## Звіт з Навчальної практики з WEB-програмування
## Тема: Гра “Хрестики-нулики”

Мета: Створити просту браузерну гру, де двоє гравців можуть по черзі ставити символи (Х та О) на полі 3х3 до перемоги одного з них.
Проект складається з трьох основних частин:
Структури (HTML)
Оформлення (CSS)
Логіки(JavaScript)
1. HTML (Файл Syte.html)
Створення контейнера, де розміщений заголовок та два екрани: Меню та сама гра.
В меню є 2 кнопки для вибору режиму: звичайна гра або до 3-ьох перемог.
Ігрове поле зроблено за допомогою 9 блоків div з класом cell. Кожен блок має свій номер, щоб програма розуміла, куди саме натиснув гравець.
2. CSS (Файл style.css)
Щоб гра виглядала сучасно, я обрав темну тему та темно-сірий фон.
Хрестики підсвічуються блакитним кольором, а нулики рожевим.
Додано ефект наведення кнопки та клітинки.
3. JavaScript (Файл script.js)
Функція tap(i): Спрацьовує, коли хтось клікає на клітинку. Вона перевіряє, чи порожнє місце, ставить туди символ поточного гравця і змінює хід.
Функція checkWin(): Це перевірка на перемогу. Вона знає 8 виграшних комбінацій (горизонталі, вертикалі та діагоналі) і після кожного ходу перевіряє, чи хтось не виграв.
Режими гри: Програма вміє рахувати бали. Якщо обрано режим «До 3-ьох», вона запам'ятовує рахунок і оголошує чемпіона, коли хтось набирає 3 перемоги.
Функція resetBoard(): Дозволяє очистити поле для нового раунду без перезавантаження всієї сторінки.
Висновок: У ході роботи я створив повністю функціональну браузерну гру «Хрестики-нулики». Проект складається з трьох роздільних файлів, що дозволило чітко розмежувати структуру, зовнішній вигляд та логіку гри.
